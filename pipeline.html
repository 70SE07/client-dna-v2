ДНК Клиента v2 — Пайплайн

Визуализация потоков, модулей и user cases

ЦКП системы
Готовый оффер (или матрица офферов) для рекламы, лендинга, воронки
Вход
От темы/ниши (ноль данных) до сырых данных (кастдевы, отзывы)
Выход
Документ-фундамент: ЦА + боли + барьеры + трансформация + триггеры + оффер(ы)
Режимы
Автоматический / Управляемый / Гибрид
